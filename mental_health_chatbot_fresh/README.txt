# Mental Health Chatbot
Simple college project built using CrewAI, Streamlit, and Google Gemini Pro.

## How to setup:
1. Extract the zip file.
2. Open terminal inside the folder.
3. Install required packages:
   pip install streamlit crewai
4. Open `main.py` and change `my_key = "PASTE_YOUR_GEMINI_API_KEY_HERE"` with your real Gemini API key.
5. Run the app:
   streamlit run main.py
